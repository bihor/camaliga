This example shows a Bootstrap Collapse/Accordion. Note 1: you need Bootstrap & Bootstrap Collapse for this example. Download it from here: https://getbootstrap.com/docs/4.2/components/collapse/ Copy the JS- and CSS-file into your fileadmin-folder! Or better: install a Bootstrap extension and you are done! Note 2: if you want to copy this HTML-file to your fileadmin-folder, you can specifty the folder like this: plugin.tx_camaliga.view.templateRootPaths.1 = fileadmin/template/files/ Copy this file in a subfolder named "Content" of this folder.
