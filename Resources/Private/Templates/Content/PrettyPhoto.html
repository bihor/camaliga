This template is a gallery that requires jQuery and prettyPhoto from here: http://www.no-margin-for-errors.com/projects/prettyphoto-jquery-lightbox-clone/documentation Download it and include it like this with TypoScript: page { includeJSFooter.prettyphoto = fileadmin/Resources/Public/Scripts/jquery.prettyPhoto.js includeCSS.prettyphoto = fileadmin/Resources/Public/Css/prettyPhoto.css } Start it with: page.jsFooterInline.810 = TEXT page.jsFooterInline.810.value = $(document).ready(function(){ $("a[rel^='prettyPhoto']").prettyPhoto({theme: 'facebook',slideshow:5000, autoplay_slideshow:true}); }); Note: if you want to copy this HTML-file to your fileadmin-folder, you can specifty the folder like this: plugin.tx_camaliga.view.templateRootPaths.1 = fileadmin/template/files/ Copy this file in a subfolder named "Content" of this folder.
