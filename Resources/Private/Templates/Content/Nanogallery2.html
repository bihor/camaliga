{namespace cam=Quizpalme\Camaliga\ViewHelpers} jQuery nanogallery2 example. You need the nanogallery2 plugin from here for this example: https://nanogallery2.nanostudio.org/ Extract the zip file and copy it to your fileadmin-folder. Include the CSS- and JS-file with TypoScript (or here) from your fileadmin-folder. Note: if you want to copy this HTML-file to your fileadmin-folder, you can specify the folder like this: plugin.tx_camaliga.view.templateRootPaths.1 = fileadmin/template/files/ Copy this file in a subfolder named "Content" of this folder.
